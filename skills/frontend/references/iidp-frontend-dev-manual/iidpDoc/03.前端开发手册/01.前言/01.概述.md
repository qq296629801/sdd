## 1.平台简介

谷神工业数字平台，是以工业软件为背景，对各产品业务进行一系列归纳整理总结出来的适用于大部分场景的标准模板页面，具备统一的标准。可以根据不同视图配置，实现不同模版的切换，以满足不同的需求场景。


## 2.文档目的

- 1.通过谷神工业数字平台**视图配置**的学习，快速实现常见页面的视图配置

- 2.通过谷神工业数字平台**扩展**的学习，可自定义组件、自定义指令，快速实现特殊需求的实现

## 3.技术栈与版本约束

平台标准前端底座与渲染引擎基于 Vue 2.7（Vue2 体系）。因此：

- 自定义组件需以 Vue 2.x 组件规范开发（以 Options API 为主），并按平台要求完成注册与引入。
- 引入第三方 UI 组件库时，必须选择支持 Vue 2.x / Vue 2.7 的版本；仅支持 Vue 3 的 UI 组件库不适用。
- 如需使用 Vue 2.7 提供的 Composition API 或相关语法，请先确认当前工程构建链与平台底座已启用支持；默认以 Vue 2.x 常规写法为准。

## 4.与传统模式的区别

1.开发模式
- 平台：通过后端视图配置，经过前端引擎渲染成标准模板页面，以“最少的手写代码”的方式搭建应用系统。
- 传统：采用“全程手写代码”的方式搭建应用系统

2.目标开发者
- 平台：对开发者技术要求不高，可根据文档配置视图构建页面
- 传统：需要熟练掌握多种编程语言和框架，因此对开发者的技术要求很高

3.应用场景
- 平台：适用于不太复杂的业务管理层应用
- 传统：针对复杂的、个性化的业务决策层应用
