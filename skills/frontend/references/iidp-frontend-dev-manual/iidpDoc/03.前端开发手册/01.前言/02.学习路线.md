# 学习路线

- 1.学习 **[环境准备](/pages/de67e3/)** 模块：搭建环境，运行demo，熟悉平台标准模板功能

- 2.学习 **[工程说明](/pages/3a50a2/)** 模块：结合demo，了解工程结构及功能

- 3.学习 **[组件](/pages/810e65/)** 模块：了解平台组件及组件配置

- 4.学习 **[在线视图](/pages/961578/)** 模块：熟悉各个场景的视图配置，并在开发者中心/视图，尝试配置

- 5.学习 **[框架](/pages/ff8358/)** 模块：了解平台框架，熟悉功能，运行机制

- 5.学习 **[扩展说明](/pages/a62bee/)** 模块：熟悉扩展流程，尝试扩展

- 6.自定义封装组件：详见 组件模块/**[自定义vue组件](/pages/587afe/)** 

- 7.自定义封装指令：详见 框架模块/**[自定义指令协议](/pages/4df791/)**



## 常见概念

1.视图：描述页面的JSON配置（详见 **[视图](/pages/961578/)**）
- 前端视图：描述页面结构、节点属性、交互操作的JSON配置
- 后端视图：根据不同模型定义的可以转换成前端视图的JSON配置

    [前后端视图区别及配置.pptx](/iidpdoc/file/前后端视图区别及配置.pptx)

2.标准模板：平台统一的，适用于大部分场景的标准模板页面（详见 **[标准模版](/pages/cf03d9/)**）

3.扩展：对标准模板以外的功能开发（详见 **[扩展说明](/pages/a62bee/)**）

4.模型：对事物的抽象化描述（详见 **[模型](/pages/0dc273/)**）

5.服务：类似与传统开发中的api接口
