# 平台提示语插件
方便项目在使用vscode开发时，提示语插件会自动提示平台的公共方法、视图组件、标准视图，方便开发


# vscode 离线安装插件

# 提示语插件下载
[提示语插件下载](/iidpdoc/file/t-plugin-0.0.4.vsix)

# 离线安装
vscode 点击扩展菜单图标：在扩展视图中，点击右上角的菜单按钮（三个点），然后选择“Install from VSIX”。
找到对应的 .vsix 文件安装（上文的提示语插件下载）

![](/iidpdoc/img/tipsU.png)



# 使用

插件安装后语法提示会自动生效

vscode 使用：

按了点. 出了下拉列表 选中按回车 才会触发，按 tab 还是可以触发 ai 编辑器的补全
