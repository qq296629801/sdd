## 通过 pre 复用到多个节点

在 `selector` 中配置 `pre` 字段，可将同一个扩展复用到多个 id 前缀对应的节点上。

**`pre` 支持三种写法：**

| 写法 | 说明 |
|------|------|
| 字符串 | 单个前缀，如 `'prexx_'` |
| 数组 | 多个前缀，如 `['prexx1_', 'prexx2_']` |
| 函数 | 动态返回前缀数组，签名为 `(extendName, config) => string[]` |

参考示例：[demo 清单 - 扩展侧边栏](/pages/3a50a2/#_11、扩展侧边栏)

```js
export default {
  'sidebar_extend_view': {
    type: 'before',
    selector: {
      attr: 'id',
      // 字符串写法：复用到 prexx_sidebar_about_button
      // pre: 'prexx_',

      // 数组写法：复用到 prexx1_sidebar_about_button、prexx2_sidebar_about_button
      pre: ['prexx1_', 'prexx2_'],

      // 函数写法：动态返回前缀数组
      // pre: (extendName, config) => {
      //   // extendName：当前扩展名，此处为 'sidebar_extend_view'
      //   // config：当前扩展配置对象
      //   return ['prexx1_', 'prexx2_'];
      // },

      value: 'sidebar_about_button'
    },
    view: {
      type: 'text',
      value: 'Hello World',
      style: { color: '#07ff00' }
    }
  }
};
```
> 上面相当于把扩展复用到 prexx1_sidebar_about_button  prexx2_sidebar_about_button 两个节点上

## 通过公共方法复用逻辑

将扩展逻辑封装到公共 JS 方法中，通过 `import` 引入后在 `beforeOperate` 中调用。可借助 `customOptions` 对不同节点进行差异化定制。

**扩展文件：**

```js
import { myCommonFn } from './myCommonFn';

export default {
  xxxx_expand: {
    type: 'replace',
    selector: {
      attr: 'id',
      pre: ['prexx1_', 'prexx2_'],
      value: 'my_btn'
    },
    beforeOperate: (app, operateItem, options) => {
      // 针对特定节点的差异化定制
      const customOptions = {
        'prexx1_my_btn': {
          xxcustomHook: (res) => {
            res.aa = 2;
            return res;
          }
        }
      };
      operateItem.view = myCommonFn(app, operateItem, options, customOptions[operateItem.selector.value] || {});
      return operateItem.view;
    },
    view: {}
  }
};
```

**公共方法 `myCommonFn.js`：**

```js
const myCommonFn = (app, operateItem, options, customOptions = {}) => {
  let reView = {
    type: 'button',
    aa: 1
  };
  // 若传入了定制钩子则执行
  if (customOptions?.xxcustomHook) {
    reView = customOptions.xxcustomHook(reView);
  }
  return reView;
};

export { myCommonFn };
```

上面通过 `pre` 将扩展复用到 `prexx1_my_btn` 和 `prexx2_my_btn` 两个节点，同时通过 `customOptions.xxcustomHook` 对 `prexx1_my_btn` 节点进行差异化定制（将 `aa` 改为 `2`），实现复用与定制的分离。
