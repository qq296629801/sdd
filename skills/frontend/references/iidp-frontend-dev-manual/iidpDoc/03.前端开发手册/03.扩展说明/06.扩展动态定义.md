## 概述

扩展动态定义允许在运行时（如生命周期钩子中）动态注册扩展，而非在配置阶段静态声明。

## tech_app.setExtend

**签名：**

```js
tech_app.setExtend('扩展名', { type, selector, ...扩展定义对象 })
```

- `tech_app` 是 app 全局变量，也可替换为上下文中的 `app`（如 `vm.app`）
- 第一个参数为扩展名（全局唯一）
- 第二个参数与静态扩展定义结构一致

**示例：** 在按钮节点的 `created` 生命周期中动态注册一个扩展，为 `table_toolbar_create` 节点合并新属性：

```js
{
  type: 'button',
  id: 'button_123',
  created: (vm) => {
    tech_app.setExtend('my_custom_extend_view_123456', {
      type: 'merge',
      selector: {
        attr: 'id',
        value: 'table_toolbar_create'
      },
      view: {
        newAttr: 'newAttr'
      }
    });
  }
}
```

当 `button_123` 节点触发 `created` 生命周期时，动态创建扩展 `my_custom_extend_view_123456`，将 `newAttr` 属性合并到 `table_toolbar_create` 节点上。
