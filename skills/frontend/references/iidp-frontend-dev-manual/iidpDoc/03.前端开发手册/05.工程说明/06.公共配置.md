# 公共配置

前端应用公共配置，包括版本配置、应用logo、名称、页面路由、公共样式、公共方法、公共主题、公共数据源等配置

## common.js
./apps/[自己的扩展应用]/common/common.js
```js
export default {
  "version": "1.0.0", // 应用版本
  "config": { // 应用公共配置
    // 'appName': 'IIDP', // 应用名称
    'appDescription': '', // 应用描述
    'logo': async () => { // 应用logo图标
      const v = await import('./img/favicon.ico');
      return v.default;
    },
    'historyMode': 'hash', // 浏览器路由：browser  哈希路由：hash
    'keywords': '', // 页面seo关键字 多个关键字以逗号隔开
    'description': '', // 页面seo描述 多个描述内容以逗号隔开
    'designWidth': 1920, // 按需重置设计稿宽度
    // 改变布局适配计算宽度 例如限制最小计算宽度为1200
    'changeCalcWidth': (clientWidth) => {
      // console.log(' === window.devicePixelRatio == ', window.devicePixelRatio);
      // console.log(' === clientWidth == ', clientWidth);
      // console.log(' === designWidth == ', this.designWidth);
      if (window.navigator.userAgent.match(/(phone|pad|pod|iPhone|iPod|ios|iPad|Android|Mobile|BlackBerry|IEMobile|MQQBrowser|JUC|Fennec|wOSBrowser|BrowserNG|WebOS|Symbian|Windows Phone)/i)) {
        // 移动端
        // clientWidth = Math.min(800, clientWidth);
        clientWidth = 1920 / 1.8;
      } else {
      // pc端
        clientWidth = 1920; // / window.devicePixelRatio;
        document.body.style.minWidth = '1000px';
      }
      return clientWidth;
    }
  }
}
```
