# 自定义工程化配置

使用场景如：新增解释器jsx-loader，可在build\webpack.common.js文件中增加配置

```js
// build\webpack.common.js
// webpack工程打包 公共配置
const { merge } = require('webpack-merge');
const commonMerge = require('@tech/t-build/BuildWebpack/webpack.common.merge.js');
module.exports = merge(commonMerge, {
  // 此处添加对应的工具配置
});

```
