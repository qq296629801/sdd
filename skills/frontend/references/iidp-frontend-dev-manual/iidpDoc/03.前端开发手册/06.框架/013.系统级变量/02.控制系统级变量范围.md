# 控制系统级变量范围

## 系统级变量

在工程的 `config/apps.json` 中配置的功能变量为**系统级变量**，配置后会作用到整个运行的前端，包括其中所有的 app。

示例：[主表单详情预览状态展示](/pages/718bde/#主表单详情预览状态展示)

```json
// config/apps.json
{
  。。。
  "global": {
    "previewForm": true
  }
}
```

## app 中控制系统级变量作用范围（可选配置）

在某 app 的 `/apps/xxApp/config/app.json` 中配置，可将系统级变量的生效范围限定到指定的 url 路径前缀。

### 包含配置

配置格式：`"$url_系统变量名": ["运行时 url 路径前缀字符串 1", "运行时 url 路径前缀字符串 2"]`

例如，当前点进某功能菜单入口，运行时 url 为：

`http://oracle.snest.com:31815/iidp/base/base_overview_menu?a=1#b=1`

其中 url 路径为 `/base/base_overview_menu`（不包含域名、routerBase `/iidp/`、search 参数、hash 参数）。

在 `xxApp` 中配置 `"$url_previewForm": ["/base/"]`，则 `previewForm` 表单预览功能仅在 `/base/` 开头的 url 路径上生效。`"$app_previewForm": "true"` 只对当前 app 生效，在其他 app 中可配置不同的值。

```json
// /apps/xxApp/config/app.json
{
  。。。
  "global": {
    "$url_previewForm": ["/base/"],
    "$app_previewForm": "true"  //app级变量
  }
}
```

### 排除配置

配置格式：`"$url_exclude_系统变量名": ["运行时 url 路径前缀字符串 1", "运行时 url 路径前缀字符串 2"]`

**排除配置的优先级高于包含配置。**

例如，在 `xxApp` 中配置 `"$url_exclude_previewForm": ["/base/developer/ui-view"]`，则 `previewForm` 表单预览功能在 `/base/developer/ui-view` 开头的 url 路径上不生效。

```json
// /apps/xxApp/config/app.json
{
  。。。
  "global": {
    "$url_exclude_previewForm": ["/base/developer/ui-view"]
  }
}
```

### 多 app 范围配置的合并规则

多个 app 对同一个系统级变量进行范围控制时，配置数组会**去重并合并**。

例如，app1 配置了：

```json
// /apps/xxApp/config/app.json
{
  。。。
  "global": {
    "$url_previewForm": ["/base/"]
  }
}
```

app2 配置了：

```json
// /apps/xxApp/config/app.json
{
  。。。
  "global": {
    "$url_previewForm": ["/API/", "/base/"]
  }
}
```

则最终运行时会合并成：`"$url_previewForm": ["/base/", "/API/"]`

## 工程 apps.json 配置覆盖 app 的控制（可选配置）

当 app 里面配置了 `$url_系统变量名` 时，工程级 `apps.json` 可以进一步控制范围。

### 包含覆盖

配置 `$url_global_系统变量名`，会与各 app 的包含配置做数组合并；若不配置则视为全包含。

```json
// config/apps.json
{
  。。。
  "previewForm": true,
  "$url_global_previewForm": ["/base/developer/data"] // 会跟app的包含配置做数组合并 若不配置则全包含
}
```

### 排除覆盖

配置 `$url_global_exclude_系统变量名`，会与各 app 的排除配置做数组合并；排除配置的优先级高于包含配置。

```json
// config/apps.json
{
  。。。
  "previewForm": true,
  "$url_global_exclude_previewForm": ["/API/custom_document"] // 会跟app的排除配置做数组合并
}
```
