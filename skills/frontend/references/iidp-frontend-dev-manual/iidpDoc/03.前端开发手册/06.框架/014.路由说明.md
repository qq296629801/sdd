## 页面跳转

平台内页面跳转支持以下方式：

**方式一：`window.open`**

```javascript
// 新标签页打开
window.open('/iidp/?pageId=custom_id', '_blank')
```

**方式二：模拟 a 标签点击（target="_blank"）**

```javascript
// 模拟 a 标签新标签页打开
const a = document.createElement('a')
a.href = '/iidp/?pageId=custom_id'
a.target = '_blank'
document.body.appendChild(a)
a.click()
document.body.removeChild(a)
```

**方式三：`goTo` 方法（平台封装）**

菜单跳转使用平台提供的 `goTo` 方法，详细用法参考文档： [goTo方法使用](/pages/e80dbe/#goto-%E8%B7%B3%E8%BD%AC%E9%A1%B5%E9%9D%A2)


## 页面间的参数传递


**方式一：URL 问号参数传递**

将参数拼接在跳转 URL 的后面，通过 `?key=value` 的形式传递。

**跳转时传参示例：**

```javascript
// 跳转到目标页面并携带参数
window.location.href = '/iidp/?pageId=custom_id&id=123'
```

**目标页面 created 接收参数示例：**

```javascript
// 解析 URL 参数
created() {
  function getQueryParam(key) {
    const params = new URLSearchParams(window.location.search)
    return params.get(key)
  }

  const pageId = getQueryParam('pageId') // 'custom_id'
  const id = getQueryParam('id')         // '123'
}
```

---

**方式二：sessionStorage 传递** 

将参数存入 `sessionStorage`，在目标页面读取。适合传递较复杂或较长的数据。

**跳转前存储参数示例：**

```javascript
// 存储参数
sessionStorage.setItem('pageParams', JSON.stringify({ id: 123 }))

// 跳转页面
window.location.href = '/iidp/?pageId=custom_id'
```

**目标页面 created 读取参数示例：**

```javascript
// 读取参数
created() {
  const params = JSON.parse(sessionStorage.getItem('pageParams'))
  const id = params.id         // 123
}
```

> **注意：** sessionStorage 仅在当前浏览器标签页内有效，标签页关闭后数据自动清除。
