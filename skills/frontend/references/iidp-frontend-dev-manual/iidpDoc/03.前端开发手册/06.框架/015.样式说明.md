# 页面样式说明

## 一、单位配置说明

本项目支持两种尺寸单位配置方式：**rem** 和 **px**，可根据项目需求灵活选择。

### rem 配置规则

| 配置项 | 说明 |
|--------|------|
| 根字体大小 | `1rem = 100px` |
| 换算示例 | `4rem = 400px` |

> `<html>` 标签设置了 `style="font-size: 100px"`，`<body>` 标签设置了 `style="font-size: 14px"`，即基准字号为 100px，业务字号为 14px。


## 二、最小宽度限制

| 属性 | 值 |
|------|----|
| 默认最小宽度 | `1000px` |
| 小于最小宽度时 | 自动出现横向滚动条 |

> 对应 `<body>` 的内联样式：`min-width: 1000px`


## 三、登录框样式示例

以下为登录卡片容器的关键样式，单位均使用 rem：

```css
/* 登录卡片容器 */
element.style {
  width: 4rem;             /* 400px */
  border: 0.01rem solid rgb(204, 204, 204);
  color: rgb(139, 0, 37);
  border-radius: 0.3rem;
  font-size: 0.14rem;
  height: 0.4rem;
  margin: 0rem;
  background-color: rgb(255, 255, 255);
}
```
