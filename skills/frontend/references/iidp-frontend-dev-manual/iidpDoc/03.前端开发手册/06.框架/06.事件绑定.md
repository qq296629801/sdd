# 事件绑定
`bind_on_事件名称`，事件名必须是vue组件内存在的可主动触发的方法。
也可以自定义事件，但是需要在created生命周期触发时自行通过vm.instance改写vue组件实例，暴露相应的方法

```js
{
    type: 'container',
    items: [
        {
            type: 'button',
            bind_on_click: (params) => {
            // params.value 组件内部 click 方法的返回值
            // params.self 当前节点的实例vm
            // params.self.$select('prev') 选择前一个节点实例
            }
        }
    ]
}
```
解析：
vue组件：
``` vue
<template>
  <!-- :class="config.className" -->
  <el-button
    class="tech-button"
    :style="style"
    @click="click"
    :title="title"
    :circle="innerOptions.circle ?? (options.circle || false)"
    :disabled="innerDisabled === undefined ? options.disabled || null : innerDisabled"
    :type="innerOptions.type ?? (options.type || 'primary')"
    :size="innerOptions.size ?? (options.size || 'small')"
    :icon="innerOptions.icon ?? (options.icon || null)"
    :plain="innerOptions.plain ?? (options.plain || null)"
    :loading="loading"
    @keyup.space.native.prevent="fun"
  >
  </el-button>
</template>


```

click作为变量传给el-button组件，在组件中通过click方法返回值作为bind_on_click的参数

``` js
export default {
  name: 'tech-button',
   computed: {
    actualActions() {
      return [
        typeof this.preAction === 'function' ? this.preAction : () => true,
        ...this.actions,
        typeof this.postAction === 'function' ? this.postAction : () => true
      ]
    },},
	
  methods: {
    async click(event) { 
      for (const fn of this.actualActions) {//执行actualActions内的可执行方法
        if (typeof fn === 'function') {
          const ret = await fn(this, { model: this.model })
          if (ret === false) {
            break
          }
        }
      }
      this.$emit('clickOptionBtn', this.config, this.config.scope, this) //通过事件传值给父组件
      
	  return { // return值就是bind_on_click的参数
        name: this.name,
        model: this.model?.[this.name],
        ref: this,
        scope: this.config.scope,
        event 
      }
    },
  }
}

```
