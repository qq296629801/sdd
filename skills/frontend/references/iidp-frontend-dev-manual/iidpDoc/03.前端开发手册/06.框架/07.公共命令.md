# 公共命令
公共命令commands是可继承且可覆盖的，通过$cmd调用。
```js
{
	type: 'container',
	commands: { // 声明命令，主要作为公共方法使用，注入上下文中，可继承父级命令
		aa: (params) => {}    // 通过$cmd.aa() 在视图函数内调用, params是必传参数
	}
	items: [
		{
			type: 'input',
			bind_on_change: (params) => {
				params.self.$cmd.aa(params)
			}
		}
	]
}
```
## 命令继承和覆盖
```js
{
	type: 'container',
	commands: {
		a: () => {console.log('a')},
		b: () => {console.log('b')}
	},
	items: [
		{
			type: 'container',
			commands: {
				a: () => {console.log('a111')}, // 覆盖父级命令a
				c: () => {console.log('c')}
			},
			items: [
				type: 'container'
				created: (vm) => {
				console.log(vm.$cmd.a(), vm.$cmd.b(), vm.$cmd.c())
						// 'a111', 'b', 'c'
				}
			]
		}
	]
}
```
