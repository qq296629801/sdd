# 视图行为协议 - view

> 视图行为协议 `view` 属性，里面可以定义 `click` 点击事件（可以配置 `mouseover` 等 dom 事件名），在点击后加载视图（事件的值对象就是对应的视图），并在页面上出现新视图界面；
> 也可以定义 `mouseover` 等事件行为，鼠标移动在元素上后加载新视图。
>
> **视图挂载规则：**
>
> - 若配置 `__parentId`：则挂载到对应节点的 `items` 里面；
> - 若未配置 `__parentId` 且当前节点有定义 `items` 属性：则新视图追加到当前节点 `items` 孩子列表上；
> - 若未配置 `__parentId` 且当前节点未定义 `items` 属性：则将新视图追加到当前页面 page 的 `items` 孩子上。

```js
	type: 'container',
	view: {
		click: { // 新视图节点定义
            type: 'dialog', // 弹窗
            title: '', // 弹框标题，非必填，默认为空
            width: '', // 宽度，非必填，默认80%
            height: '', // 弹框高度，非必填，默认400px
            marginTop: 100, // showType为dialog时有效，未配置时默认130
            tableHeight: '', // 弹框中表格高度，非必填
            mainTableHeight: '', // 弹框中主表格高度，非必填，默认200px
            erTableHeight: '', // 弹框中子表格高度，非必填，默认200px
            hasFooter: true, // 弹框底部按钮，非必填，默认true
            showConfirm: true, // 弹框确认按钮，非必填，默认true
            showCancel: true, // 弹框取消按钮，非必填，默认true
            useAdvancedTable: true,  // openView默认无法使用高级表格功能，配置此项后可以使用
          	// style: { 'z-index': '3000 !important' },  // 可以重写样式  (可选配置)
		  	// 指定父级id 则为挂载到哪个节点的items里面 (可选配置)
		  	// __parentId: 'xxx_container_page_bottom', //（__parentId 注意双下划线，否则会追加节点错误）
		  	// app: 'xxxx', // (可选配置) openView的app
		  	// _noCache: true, // (可选配置) 是否不缓存上次打开数据
		  	// inheritApp: true, // (可选配置) 是否强制继承行为节点的调用接口app参数，方便做分布式 默认false
		  	// _reInit: true, // (可选配置) 是否下次打开重新注册前端视图节点
		  	// inheritData: false, // (可选配置) 不继承父级的数据源 (不配置默认是继承的)
		  	// beforeOperate: async (params) => { // openView配置前置钩子函数 return true则继续执行 false则不执行
          	//   console.log(' ==== params == ', params);
          	//   return true;
          	// },
		  	// mounted: (vm) => {
		  	//	console.log(' === 发起行为节点id vm.$ds.actionNodeId == ', vm.$ds.actionNodeId)
		  	// },
          items: []
		}
	},
	items: []
  items: []
}
```

## view 协议行为参数

`view` 是一个对象，key 为事件名，value 为“要加载的视图节点定义”。

| 属性名  | 说明                                      | 类型   | 是否可选 |
| ------- | ----------------------------------------- | ------ | -------- |
| click   | 点击时触发（可配置 mouseover 等其他事件） | object | 可选     |
| created | 节点创建时触发                            | object | 可选     |
| mounted | 节点渲染后触发                            | object | 可选     |

## 视图节点定义（例如 `click: { ... }`）的通用参数

| 属性名        | 说明                                                                      | 类型     | 是否可选 |
| ------------- | ------------------------------------------------------------------------- | -------- | -------- |
| \_\_parentId  | 指定挂载的父级节点 id（注意是双下划线，可配置触发 openView 的节点 id）    | string   | 可选     |
| \_noCache     | 是否不缓存上次打开数据。默认 `false`（缓存），配置 `true` 则不缓存        | boolean  | 可选     |
| \_reInit      | 是否每次进入都重新构建视图。默认 `false`（缓存），配置 `true` 则重新构建  | boolean  | 可选     |
| inheritApp    | 是否强制继承行为节点的调用接口 app 参数，方便做分布式。默认 `false`       | boolean  | 可选     |
| inheritData   | 是否继承父级的数据源。默认 `true`（继承）                                 | boolean  | 可选     |
| beforeOperate | 节点渲染前触发的钩子函数，返回 `true` 继续弹出视图，返回 `false` 则不弹出 | function | 可选     |

> **⚠️ 注意事项：**
>
> - `__parentId` 必须是双下划线：写成 `_parentId` 会导致追加节点错误，挂载位置不符合预期。
