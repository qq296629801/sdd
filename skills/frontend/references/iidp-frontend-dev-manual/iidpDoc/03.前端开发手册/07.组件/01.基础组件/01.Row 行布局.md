# 分栏布局

> 通过基础的 24 分栏，迅速简便地创建布局。

## 基本用法

```js
{
  type: 'row',
  span: 4, //每列用相同的span
  gutter:20,
  selfAdaptionW:false,
  items:[
    {
      type:'text',
      span:12//子节点自己控制栅格列数
    }
  ]
}
```

## Attributes

| 属性名        | 说明                                  | 类型    | 默认值                                  |
| ------------  |------------------------------------- |-------- |--------------------------------------- |
| type          | 组件名称(固定配置不可修改)                 |  string |  row                                   |
| sapn          | 栅格占据的列数                         |  number | 24/items.length                       |
| gutter        | 栅格间隔                              | number   | 20                                    |
| labelWidth    | label的宽度（嵌套在form里面时使用）     | string   | -                                     |
| selfAdaptionW | 自动计算列栅格占据列数                  | boolean  |  在form中默认true,单独使用时默认false  |
| minWidth      | 列的最小宽度，selfAdaptionW为true时有效 | boolean  |  -                                    |
