# 选择器

> 当选项过多时，使用下拉菜单展示并选择内容。

![](./img/select.png)

## 基本用法

```js
{
  type: 'select',
  name: 'select',
  text: '选择器',
  options: [{text: 'a', value: 1}, {text: 'b', value: 2}],
  multiple: false,
  clearable: true, // 是否可清空 下拉组件选择空值 配置清空兼容处理 *最佳推荐
  model:{//默认值，脱离表单单独使用该组件时使用 否则会导致表单整体update异常
    select:1//该属性名需要与name属性的值一致，默认值支持String、Number、Boolean类型
  },
  // 绑定change事件
  bind_on_changeHandler: (data) => { console.log(data) },
}


// 分组
{
  {
  type: 'select',
  name: 'select',
  text: '选择器',
  model:{//默认值，脱离表单单独使用该组件时使用 否则会导致表单整体update异常
    select:1//该属性名需要与name属性的值一致，默认值支持String、Number、Boolean类型
  },
  group: true,
  options: [{
    label: '分组1',
    options: [{
      label: 'a',
      value: 'a'
    }]
  },{
    label: '分组2',
    options: [{
      label: 'b',
      value: 'b'
    }]
  }]
}
}
```

## 使用数据源 获取 option

```js
{
  type: 'select',
  name: 'select',
  text: '选择器',
  model:{//默认值，脱离表单单独使用该组件时使用 否则会导致表单整体update异常
    select:1//该属性名需要与name属性的值一致，默认值支持String、Number、Boolean类型
  },
  // bind指令，绑定数据源数据
  bind_options: '$ds.data1',
  multiple: false,
  clearable: true,
  ds_config: {
		type: 'meta',  // 调用元模型引擎API获取数据
		name: 'data1', // 数据源标识，获取数据后将根据此名称存入dataSource.data1中
		autoRequest: true, // 在节点渲染后，是否自动触发请求，默认false
		options: {
			params: { // 参考元模型api传参规范
				args: {
					properties: ['name']
				},
				service: 'search',
				model: 'xxx'
			}
		},
    // 有必要可以在这里进行数据格式化
    reqAfter: (vm, res, config) => {
      // res 是接口返回值，假设res是数组
      let newRes = res.map(item => {
        return {
          // 显示值
          text: item.name,
          // 唯一标识
          value: item.id
        }
      })
      // 返回转换后的数据
      return newRes
    }
	}
}
```

## 使用 getData 获取 option

在表格内使用 select 时，如果查询下拉值的接口参数需要其他列字段作为参数，可以使用 getData 方法

```js
{
  type: 'select',
  prop: 'holdSiteId',
  label: '预Hold站点',
  name: 'holdSiteId',
  getData: async ({ config }) => {
    console.log(config.row); // config.row: 当前行数据
    const res = await window.Tech.httpMeta({
      ...
    });
    return res; // res格式：{ items: [{ text: 'hhhhhhhhh', value: '048r321321' }] } 需要返回items数组
  }
}
```

## Attributes

| 属性名       | 说明                                              | 类型    | 默认值 |
| ------------ | ------------------------------------------------- | ------- | ------ |
| multiple     | 是否多选                                          | boolean | -      |
| collapseTags |  多选时是否将选中值按文字的形式展示                 |boolean | false  |
| clearable    | 是否可以清空选项                                  | boolean | false  |
| options      | 选项，具体看下表                                  | array   | -      |
| value-key    | 作为 value 唯一标识的键名，绑定值为对象类型时必填 | string  | value  |
| allow-create | 是否允许用户创建新条目，需配合 filterable 使用    | boolean | false  |
| remote       | 是否为远程搜索                                    | boolean | false  |
| group        | 是否分组                                          | boolean | false  |


### Option Attributes

| 属性名          | 说明               | 类型                 | 默认值 |
| --------------- | ------------------ | -------------------- | ------ |
| value           | 选项的值           | string/number/object | -      |
| text            | 选项的标签         | string/number        | -      |
| disabled        | 是否禁用该选项     | boolean              | false  |
| defaultSelected | 是否默认选中第一项 | Boolean              | false  |

## Events

| 事件名称      | 说明             | 回调参数       |
| ------------- | ---------------- | -------------- | ------- |
| changeHandler | 在选项变更时触发 | (value: string | number) |
