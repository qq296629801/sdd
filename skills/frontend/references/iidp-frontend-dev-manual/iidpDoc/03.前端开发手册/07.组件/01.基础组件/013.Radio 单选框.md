# 单选框

![](././img/checkbox.png)

> 在一组备选项中进行单选,由于选项默认可见，不宜过多，若选项过多，建议使用 Select 选择器。

## 基本用法

```js
{
  type: 'radio-group',
  name: 'radio-group',
  text: '单选框',
  radioStyle: 'default',
  options: [{text: 'a', value: 1}, {text: 'b', value: 2}],
  size: 'mini',
  // 绑定change事件
  bind_on_changeHandler: (data) => { console.log(data) },
}
```

## Attributes

| 属性名     | 说明       | 类型   | 默认值  | 可选值                |
| ---------- | ---------- | ------ | ------- | --------------------- |
| size       | 尺寸       | string | mini    | medium / small / mini |
| options    | 选项       | array  | -       | undefined             |
| radioStyle | 单选框样式 | string | default | default/button        |

## Events

| 事件名称          | 说明                                       | 回调参数                     |
| -----------------| ------------------------------------------ | ----------------------------|
| changeHandler    |  在选项变更时触发                            | (value: string | number)    |
