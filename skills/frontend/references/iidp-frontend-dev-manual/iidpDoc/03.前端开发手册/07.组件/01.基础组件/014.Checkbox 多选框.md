# 多选框

![](./img/checkboxGroup.png)

> 一组备选项中进行多选,单独使用可以表示两种状态之间的切换
## 基本用法

```js

{
  type: 'checkbox',
  name: 'checkbox',
  text: '单个'
}

{
  type: 'checkbox-group',
  name: 'checkbox-group',
  text: '多选框组',
  options: [{text: 'a', value: 1}, {text: 'b', value: 2}],
  // 绑定change事件
  bind_on_changeHandler: (data) => { console.log(data) },
}
```

## Attributes

| 属性名  | 说明             | 类型  | 默认值 |
| ------- | ---------------- | ----- | ------ |
| options | 选项，具体见下表 | array | -      |
| 待续... |                  |       |        |

### Options Attributes

| 属性名   | 说明         | 类型                      | 默认值          |
| -------- | ------------ | ------------------------- | --------------- |
| text     | 文本         | string                    | -               |
| value    | 选中状态的值 | string / number / boolean | -               |
| 其他属性 | 其他属性     | string                    | 参考 element ui |

## Events

| 事件名称          | 说明                                       | 回调参数                     |
| -----------------| ------------------------------------------ | ----------------------------|
| changeHandler    |  在选项变更时触发                            | (value: string | number)    |
