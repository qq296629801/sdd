# 面包屑

> 显示当前页面的路径，快速返回之前的任意页面。

![](./img/breadCrumb.png)

## 基本用法

```js
{
  type: 'breadcrumb',
  appName: 'IOT',
  list: [{path: '/', label: '详情'}],
  project: ''
}
```

## Attributes

| 属性名  | 说明       | 类型   | 默认值    |
| ------- | ---------- | ------ | --------- |
| appName | 应用名称   | String | -         |
| project | 项目名称   | String | -         |
| list    | 面包屑列表 | Array  | undefined |
