# 走马灯

在有限空间内，循环播放同一类型的图片、文字等内容

## 基本用法

```js
{
  type: 'carousels',
  arrow: 'always', //切换箭头的显示时机
  autoplay: true, //自动播放
  style:{
    height:'100%'
  },
  items:[{
    type: 'container',
    style:{
      height: '100%'
    }
  },
  {
    type: 'container',
    style:{
      height: '100%'
    }
  }]
}
```

## Attributes

| 属性名    | 说明               | 类型    | 可选值               | 默认值     |是否必填|
| --------- | ------------------ | -------| ------------------- | ---------- | ----- |
| type      | 组件名称    	      | string  |	carousels          | —          | 是 |
| autoplay  | 自动播放           | Boolean |                     | false      | 否 |
| direction | 展示的方向         | String  | horizontal/vertical | horizontal | 否 |
| arrow     | 切换箭头的显示时机  | String  | always/hover/never  | hover      | 否 |
| type      | 展示类型           | String  | card                | -          | 否 |
| interval  | 自动切换的时间间隔，单位为毫秒| number |	—	         | 3000       | 否 |
