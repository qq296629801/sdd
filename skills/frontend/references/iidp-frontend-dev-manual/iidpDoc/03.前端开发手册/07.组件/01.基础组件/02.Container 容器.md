# 容器

##  基本用法
> container是一个空容器，可以将其他的组件放进items中。
```js
{
  type: 'container', 
  id: 'container_main_wrap',
  style: {
    background: '#fff',
    padding: '0.15rem'
  },
  items: [
    {
      type: 'input',
      id: 'container_input',
      placeholder: '请输入'
    }
    {
      type: 'button',
      value:'提交',
      options:{
        type:'primary',
        size:'small',
        icon: 'el-icon-upload'
      },
      bind_on_click: (data) => { console.log(data, data.self) } 
    }
  ]
},
```
