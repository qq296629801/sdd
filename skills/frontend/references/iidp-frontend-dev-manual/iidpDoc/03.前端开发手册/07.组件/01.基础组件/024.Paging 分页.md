# 分页


## 基本用法
```js
{
  type: 'paging',
  id: 'paging_001',
  pageSize: 30,
  input: '1-30',
  totalCount: '...',
  selectPageSize:{
    show: true, // 显示分页n条/页下拉
    allowCreate: true, // 允许用户创建n条/页新条目
    options: [20,50,100] // 分页n条/页下拉选项
  },
  bind_on_changePageNumber: async (res) => {
    const { self: vm, value: page } = res
  }
}
```

## Attributes
| 属性名 | 说明 | 类型 | 默认值 |
| ----- |----- |----- |----- |
|pageSize |每页显示条目个数 |number |30  |
|input |表单域标签的宽度 |string |-  |
|totalCount |总条目数 |number/string |...  |
|prev |上一页是否可用 |boolean |false  |
|selectPageSize.show |显示分页n条/页下拉 |boolean |false  |
|selectPageSize.allowCreate |允许用户创建n条/页新条目 |boolean |false  |
|selectPageSize.options |分页n条/页下拉选项 |array |[30,50,100]  |

## Events

| 事件名称          | 说明                                       | 回调参数                     |
| -----------------| ------------------------------------------ | ----------------------------|
| changePageNumber |  变更页数时触发                             | (value: object)             |
