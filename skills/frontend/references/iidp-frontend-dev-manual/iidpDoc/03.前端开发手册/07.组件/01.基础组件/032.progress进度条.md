# Progress 进度条

> 用于展示操作进度，告知用户当前状态和预期。

¶线形进度条

![](./img/process.png)

## 基本用法

```js
{
  id: 'progress',
  type: 'progress',
  name: 'progress进度条',
  percentage: 40,
  style: {
    width: '600px'
  },
  loading: true
},
```

## Attributes

| 属性名      | 说明                            | 类型                      | 默认值 |
| ----------- | ------------------------------- | ------------------------- | ------ |
| percentage  | 进度值                          | number                    | 0      |
| comType     | 进度条类型                      | line/circle/dashboard     | line   |
| status      | 进度条当前状态                  | success/exception/warning | -      |
| strokeWidth | 进度条的宽度，单位 px           | number                    | 6      |
| textColor   | 进度条字体颜色                  | string                    |        |
| strokeColor | 进度条填充颜色                  | string                    |        |
| backColor   | 进度条底色                      | string                    |        |
| textInside  | 进度条显示文字内置在进度条内    | boolean                   | false  |
| showText    | 是否显示进度条文字内容          | boolean                   | false  |
| style       | 样式                            | object                    | {}     |
| width       | 环形进度条画布宽度              | number                    | 126    |
| loading     | 通过 loading 状态控制进度条显示 | boolean                   | false  |
| loadingText | 进度条显示的文字信息            | string                    | -      |

## 轮巡进度接口更新进度条

点击按钮调用接口，同时全屏显示进度条。根据接口的返回值轮巡进度接口对进度条进行更新。

```js
{
  window.Tech.httpMeta({
    data: {
        params: {
            ...
        }
    },
    progressReq: (res) => {
      // res为上一次请求返回结果
      return new Promise((resolve, reject) => {
        window.Tech.httpMeta({
          data: {
            params: {
              args: {
                excFlagId: '123'
              },
              app: 'xxxx',
              service: 'xxxx',
              model: 'xxxx'
            }
          }
        })
          .then((excProgressData) => {
            let excProgressRate = excProgressData?.data?.excProgressRate * 100;
            excProgressRate = excProgressRate.toFixed(2);
            excProgressRate = parseFloat(excProgressRate);
            .......
            resolve(excProgressRate); // 返回值必须是进度值 number类型
          })
          .catch((err) => {
            console.log(err, 'err');
            reject(err);
          });
      });
    },
    progressReqInterval: 5000 // 轮巡时间间隔 默认3000ms
})
}
```
