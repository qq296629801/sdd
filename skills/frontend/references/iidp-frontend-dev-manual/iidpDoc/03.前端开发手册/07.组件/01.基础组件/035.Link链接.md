# Link 链接

## 基本用法

```js
{
  id: 'link',
  type: 'link',
  name: 'link标记',
  value: '跳转链接',
  href: 'https://baidu.com'
},
```

## Attributes

| 属性名    | 说明         | 类型                                        | 默认值  |
| --------- | ------------ | ------------------------------------------- | ------- |
| value     | 显示的内容   | number/string                               | -       |
| linkType  | 类型         | primary / success / warning / danger / info | primary |
| href      | 链接         | string                                      | -       |
| underline | 下划线       | boolean                                     | false   |
| disabled  | 是否禁用状态 | boolean                                     | false   |
