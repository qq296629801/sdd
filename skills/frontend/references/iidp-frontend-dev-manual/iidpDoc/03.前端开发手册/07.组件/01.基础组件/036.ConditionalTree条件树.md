# 条件树

此组件依据波兰表示方式组装数据

## 配置图标

```js
{
  type: 'conditional-tree',
  name:'conditional',
  id:'conditional',
  ctrlIcons：{
    addNode: 'el-icon-circle-plus-outline',
    delNode: 'el-icon-delete',
    hideContent: 'el-icon-minus',
    showContent: 'el-icon-edit-outline',
  }
}

```

## 隐藏删除按钮

```js
{
  type: 'conditional-tree',
  name:'conditional',
  id:'conditional',
  hasDel:false,//默认true
}

```

## 自定义收起内容

```js
{
  type: 'conditional-tree',
  name:'conditional',
  id:'conditional',
  customContent:(data,config)=>{
    // console.log(data,config)
    return `<span style="color:red">214345<span>`
  },
}

```

> **注意：**
>
> 1. 此组件中的 items，仅用作渲染的模板，不是视图实例，不可以用`tech_app.page.getNode`、`vm.$select`等获取视图实例的方法获得实例
> 2. 在 form 中，form 的 value 数据结构为树状，因此获取节点块的表单需要根据数据路径（dataPath）获得在表单中对应的数据（value）
> 3. 可以根据数据路径（dataPath）获取 formItem 的 dom 实例，`document.getElementById(dataPath+'formItem的name属性')`
>    items 仅支持嵌套 row 组件，以及 form 中的输入类型的组件

## Attributes

| 属性名          | 说明                                                         | 类型     | 默认值                                                                                                                                                                                                                            |
| --------------- | ------------------------------------------------------------ | -------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| labelWidth      | label 的宽度（嵌套在 form 里面时使用）                       | string   | -                                                                                                                                                                                                                                 |
| span            | 栅格占据的列数                                               | number   | 24                                                                                                                                                                                                                                |
| name            | 数据中的字段名，必须                                         | string   | -                                                                                                                                                                                                                                 |
| model           | 绑定的数据，脱离表单时使用，表单中不可用                     | object   | -                                                                                                                                                                                                                                 |
| hasDel          | 不显示删除按钮                                               | boolean  | true                                                                                                                                                                                                                              |
| briefingStyle   | 简报的样式                                                   | -        |
| logicList       | 逻辑列表                                                     | map      | {<br/>'\|': '或',<br/>'\&': '且'<br/>}                                                                                                                                                                                            |
| ctrlIcons       | 按钮图标                                                     | map      | {<br/>addNode: 'el-icon-circle-plus-outline',//添加按钮<br/>delNode: 'el-icon-delete',//删除按钮<br/>hideContent: 'el-icon-minus',//纯文本隐藏，显示表单时<br/>showContent: 'el-icon-edit-outline',//纯文本显示，显示表隐藏<br/>} |
| conditionalData | 波兰表示的数组,至少有一个逻辑符和一个条件                    | array    | []                                                                                                                                                                                                                                |
| customContent   | 收起表单，显示文本的内容，                                   | function | 参数：<br/>data：当前节点的表单数据，<br/>config:当前节点的输入组件 config<br/>返回(必须) ：字符串，支持 html 标签                                                                                                                |
| items           | 节点的模板，目前支持嵌套的组件：row、form 中的输入类型的组件 | Array    | -                                                                                                                                                                                                                                 |

## Events

| 事件名称    | 说明             | 回调参数                                                                                                                                                                    |
| ----------- | ---------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| getResult   | 获取结果         | resultTree：树结构的结果, <br/>resultArr：波兰表示的结果                                                                                                                    |
| onChangeVal | 当改变输入时触发 | data(object)：当前节点块的数据，<br/> dataPath(string)：当前节点的数据路径，<br/> dataIndex(array):当前节点的数据路径集合 <br/> nodeEl：当前节点块的输入类型组件的配置<br/> |

## 实例事件

| 事件名称   | 说明                               | 参数                                                     |
| ---------- | ---------------------------------- | -------------------------------------------------------- | ------------------------- |
| findConfig | 获取当前节点块的输入类型组件的配置 | resultTree：树结构的结果, <br/>resultArr：波兰表示的结果 | String:当前节点的数据路径 |
