# Alert 提示

用于页面中展示重要的提示信息。

```javascript
{
  type: 'alert',
  title: '成功提示的文案',
  theme: 'success',
  effect: 'light',
  'show-icon': false,
  description: '辅助性文字'
}
```

## Attributes

| 属性名      | 说明           | 类型    | 默认值 | 可选值                     |
| ----------- | -------------- | ------- | ------ | -------------------------- |
| title       | 标题           | string  | -      | -                          |
| theme       | 主题           | string  | info   | success/warning/info/error |
| effect      | 选择提供的主题 | string  | light  | light/dark                 |
| description | 辅助性文字     | string  | -      |
| show-icon   | 是否显示图标   | boolean | false  |
| closable    | 是否可关闭     | boolean | false  |
