# 富文本

## 引入

在引入自定义组件的文件夹手动引入 common/comps.js 加入下面代码

```js
import inputRichText from '@tech/t-el-ui/biz/native/inputRichText/build.inputRichText.es.js'
import '@tech/t-el-ui/biz/native/inputRichText/style.css'

export default {
  inputrichtext,
}
```

## 基本用法

```js
       {
         type: 'inputrichtext',
         modelValue: '123',
         editorConfig:{
           placeholder: '请输入内容'
         }
         bind_on_change: (params)=> {
           console.log(params);
         }
       }
```

## 表单用法

```js
       {
          type: 'form',
          dataSource: {
            form: {
              value: 123
            }
          },
          items: [
            {
              type: 'inputrichtext',
              name: 'value',
              text: 'value'
            }
          ]
        }
```

## Attributes

| 属性名        | 说明                                                                   | 类型   | 默认值 |
| ------------- | ---------------------------------------------------------------------- | ------ | ------ |
| toolbarConfig | 请查看 [wangEditor](https://www.wangeditor.com/v5/toolbar-config.html) | object | {}     |
| editorConfig  | 请查看 [wangEditor](https://www.wangeditor.com/v5/editor-config.html)  | object | {}     |
| style         | 样式                                                                   | object | -      |

## Events

| 事件名称 | 说明             | 回调参数        |
| -------- | ---------------- | --------------- |
| change   | 绑定值改变时触发 | (value: string) |
