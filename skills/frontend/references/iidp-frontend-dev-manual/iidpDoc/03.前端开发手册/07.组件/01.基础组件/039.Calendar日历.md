# 日历

## 基本用法

```js
  {
    type: 'calendar',
    scheduleData: [
      {
        workingDay: '2023-05-01',
        content: [
          {
            notice: '提醒',
            type: 'important'
          }
        ]
      }
    ]
  }
```

## Attributes

| 属性名       | 说明         | 类型  | 默认值 |
| ------------ | ------------ | ----- | ------ |
| scheduleData | 设置某日提醒 | Array | []     |
