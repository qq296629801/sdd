# 图片组件

```js
{
  type: 'image',
  // id:'base_overview_menu_container_main_home_page_banner',
  width: '100%',
  height: 'inherit',
  src: '/static-resource/homePage/banner.png',
}
```

## Attributes

| 属性名 | 说明     | 类型   | 默认值 |
| ------ | -------- | ------ | ------ |
| width  | 宽度     | String |        |
| height | 高度     | String |        |
| src    | 图片路径 | String |        |
