# 按钮

![](./img/button.png)

## 基本用法

```js
{
  type: 'button',
  value:'提交',
  options:{
    type:'primary',
    size:'small',
    icon: 'el-icon-upload'
  },
  // 绑定按钮的点击事件
  bind_on_click: (params) => { console.log(params, params.self) }
}
```

## 调用接口

```js
{
  type: 'button',
  value:'提交',
  options:{
    type:'primary',
    size:'small',
    icon: 'el-icon-upload'
  },
  // 绑定按钮的点击事件
  bind_on_click: (params) => {
    // 调用数据源，异步方法
    params.self.request('dataSave', {});
    // 详见 框架/数据源
  },
  ds_config: {
		type: 'meta',
		name: 'dataSave', // 数据源标识
		// method: 'save' // 标识改配置是保存逻辑，调用后的结果不会缓存到dataSource中
		options: {
			params: {
				args: {
					properties: ['name']
				},
				service: 'search',
				model: 'xxx'
			}
		}
	}
}
```

## Attributes

| 属性名  | 说明               | 类型    | 默认值 |
| ------- | ------------------ | ------- | ------ | --- |
| value   | 按钮文本           | string  | -      |
| options | 配置项，具体看下表 | object  | -      |
| loading | 是否加载中状态     | boolean | false  |     |

### Options Attributes

| 属性名   | 说明         | 类型    | 默认值 | 可选值                                             |
| -------- | ------------ | ------- | ------ | -------------------------------------------------- |
| type     | 类型         | string  | -      | primary / success / warning / danger / info / text |
| size     | 尺寸         | string  | -      | medium / small / mini                              |
| icon     | 图标类名     | string  | -      |                                                    |
| disabled | 是否禁用     | boolean | false  |                                                    |
| plain    | 是否朴素按钮 | boolean | false  |                                                    |
