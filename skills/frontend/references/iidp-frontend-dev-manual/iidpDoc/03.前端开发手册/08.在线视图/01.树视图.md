# 树视图

## 树视图配置

在菜单对应的模型中配置 tree 后端视图

```json
{
  "type": "tree",
  "model": "rbac_organization", //树的数据来源
  "columns": [
    //请求树的数据字段集合
    "name",
    "parent_id",
    "children_ids",
    "description"
  ],
  "props": {
    "children": "children_ids", //树的数据中以children_ids字段为children
    "parent": "parent_id", //树的数据中以parent_id字段为parent，parent_id需要以{value:'',id:''}形式
    "label": "name", //树的数据中以name字段为label
    "showRoot": "true", //自动展开根节点
    "subApp": "base", //右侧视图loadview接口的app参数
    "subModel": "rbac_user", //右侧视图为表格时loadview接口的模型参数
    "subViewType": "grid,search,form", //右侧视图为表格时loadview接口的视图类型参数
    "subViewFilter": "org_id", //右侧视图获取数据接口的添加filter参数，字段名为org_id
    "subViewFilterValue": "id", //右侧视图获取数据接口的添加filter参数，字段名为org_id需要取当前数据字段为id的值，拼接后接口的filter:[["org_id","in",树的数据中id的值]]
    "hasIcon": true, //每个节点右侧是否有图标按钮（添加、编辑、删除）
    "lazy": true, //开启树节点懒加载
    "formSubViewModel": "xxxx", //右侧视图为表单时loadview接口的模型参数
    "formSubViewType": "xxxx_form", //右侧视图为表单时loadview接口的视图类型参数
    "isLoadViews": true, //点击树节点是否反复调用右侧视图loadView接口，应用与动态视图场景
    "isDrag": true, // 树宽度是否可进行拖拽调整 默认 true
    "customSubViewType": "xxxx_form", // 新增编辑的自定义视图类型，默认form
    "maxLevel": 10, //树新增最大层级限制
    "showTree": false, // 搜索结果是否显示树形结构，默认false
    "showSubView": false, //是否显示右侧视图，默认false
    "virtualScrollTree": true, //是否开启虚拟滚动树，默认false
    "isRemote": true, //是否开启远程数据，默认false
    "defaultExpandAll": true, //是否默认展开所有节点，默认false
    "showCheckbox": true, //是否显示复选框，默认false
  },
  "tbar": [
    "@defaults" //默认add，delete按钮，已实现添加和删除事件
    // ... 更多按钮请参考button组件的配置，其中的点击事件需要扩展实现
  ],
  "search": true //是否显示搜索
}
```

## 回填主表单中字段（外列表树选中的节点）

- subViewFormInitValues

```js
//只回填一个字段的情况
{
  "type": "tree",
  "props": {
    "children": "children_ids",
    "parent": "parent_id",
    "label": "name",
    "subViewFilter": "org_id",//回填字段
    "subViewFromDefault": "id", //树对应字段(如表单回填字段和树对应字段一致此项可不配置)
  },
}
//需回填多个字段的情况
{
  "type": "tree",
  "props": {
    "children": "children_ids",
    "parent": "parent_id",
    "label": "name",
    "subViewFormInitValues": [//回填字段
      {
        "treeName": "type",//树对应字段
        "subformName": "tag",//表单的回填字段
      },
      {
        "label":'显示名',//指定显示名,form字段类型为lookup、select时生效
        "labelInValue":true,//指定显示名,form字段类型为lookup、select时生效
        "treeName": "type",//树对应字段
        "subformName": "tag",//表单的回填字段
      }
    ],
  },
}

```

## 树结果查询自定义 search 接口

- `searchConfig:{}`
  数据类型为 Object，与接口请求参数对象一致，此参数与基础视图中请求的参数合并，组合成新的请求参数作为原来 search 接口的参数

```js
{
  "type": "tree",
  "searchConfig":{
    "model":"",
    "service":"",//service也可以修改，会在请求loadview接口后调用此接口
    "args":{
      "filter":[["id","=","1234"],["name","like","%test%"]],//只需要把条件列出，标准视图自动拼接逻辑符号，目前默认’&‘
      "properties":[]//需要请求哪些字段，此配置会与原来接口的properties合并
      // ...更多参数
    }
  }
}
```

![自定义 search 接口请求](./img/tree_searchConfig.gif)

## buttons 节点图标按钮

节点图标按钮列表,只支持新增，编辑，删除按钮

```js
{
  "type": "tree",
  "buttons":[
    {
      "action":"create",
      "parentDisabled": true  //配置新增时父级选择框是否禁用
    },
    {"action":"edit"},
    {
      "action":"delete",
      "message":"确定删除吗？"   //配置删除提示语
    }
  ]
}
```

## 控制树节点后的操作图标按钮显示

- showBtnFn
- 控制树节点后的操作图标按钮显示，包括默认 buttons 的按钮和自定义的 customButtons 的按钮
- 方法中返回对象，key 为按钮的 action，true 显示，false 隐藏

```js
{
  "type": "tree",
  "showBtnFn":(nodeData) => {
    return {
      create:true,
      edit:nodeData.id == 'aa',
      delete:!!nodeData.parent,
    }
  },
}
```

## 树节点可拖拽

- draggable: 默认 false, 配置 strue 后，树节点顺序可拖拽
- allowDrag: 控制节点是否可拖拽，方法返回 true 则可拖拽，false 则不可拖拽
- allowDrop: 控制节点是否可放置，方法返回 true 则可放置，false 则不可放置

```js
{
  "type": "tree",
  "props":{
    "draggable": true,
    // ...更多配置
  },
  "allowDrag":(draggingNode) =>{
      // draggingNode:当前拖拽节点数据
      if(draggingNode.data.name === 'eeee'){return true} else {return false}
   },
   "allowDrop": (draggingNode, dropNode, type) =>{
      // draggingNode:当前拖拽节点数据
      // dropNode:当前放置节点数据
      // type:放置节点的类型
      if(dropNode.data.name === 'ffff'){return true} else {return false}
    }
}
```
## 自定义节点操作按钮

- 2.9.0 版本开始支持
- customButtons:树节点的自定义操作按钮
- 需在 showBtnFn 方法中控制每个按钮的显示隐藏
- 需扩展实现按钮的自定义事件

```js
{
  "type": "tree",
  "props":{

    // ...更多配置
  },
  "buttons": [],
  "customButtons": :[
    {
      "action": "show",
      "name": "显示",
      "icon": "iconfont icon-mimakejian"
    },
    {
      "action": "hide",
      "name": "隐藏",
      "icon": "iconfont icon-mimabukejian"
    }]
}
```

## 配置参数

| 属性名     | 说明                                                                                                                                    | 类型     | 是否可选 | 默认值 |
| ---------- | --------------------------------------------------------------------------------------------------------------------------------------- | -------- | -------- | ------ |
| type       | 视图类型                                                                                                                                | string   | 必填     | -      |
| model      | 请求树数据的模型                                                                                                                        | string   | 必填     | -      |
| columns    | 请求树数据的字段                                                                                                                        | array    | 必填     | -      |
| props      | 包含对树的数据解析，右侧表格的请求参数                                                                                                  | array    | 可选     | -      |
| hasAllNode | 添加一个‘全部’的根节点，该节点的 id 为""                                                                                                | boolean  | 可选     | -      |
| buttons    | 节点右侧的图标按钮,只支持 create，edit，delete                                                                                          | array    | 可选     | []     |
| mergeAttrs | 配置额外属性到渲染后的树前端视图节点，方便配置自定义指令                                                                                | object   | 可选     | {}     |
| showBtnFn  | 控制每个节点的 button 显示，参数 nodeData:当前节点的数据，方法要求返回对象，如{<br/>create:true，<br/>edit:false,<br/>delete:true<br/>} | function | 可选     | -      |

## props 配置参数

| 属性名                | 说明                                                                                                     | 类型    | 是否可选 | 默认值                                                                                           |
| --------------------- | -------------------------------------------------------------------------------------------------------- | ------- | -------- | ------------------------------------------------------------------------------------------------ |
| id                    | 指定树数据中与 id 对应的字段                                                                             | string  | 可选     | id                                                                                               |
| label                 | 指定树数据中与 label 对应的字段                                                                          | string  | 可选     | name                                                                                             |
| children              | 指定树数据中与 children 对应的字段                                                                       | string  | 可选     | 1. 配有 props 时，默认 children <br/> 2.有配置 props 且没有 children 时，默认 null（没有子节点） |
| parent                | 指定树数据中与 parent 对应的字段，树数据中表示父节点的字段可以是 string，也可以是{value:'',id:''}形式    | string  | 可选     | 1. 配有 props 时，默认 parent <br/> 2. 有配置 props 且没有 parent 时，默认 null（没有父节点）    |
| subModel              | 右侧视图请求接口中的 model 字段参数                                                                      | string  | 必须     | -                                                                                                |
| subViewType           | 右侧视图请求接口中的 type 字段参数，loadview 接口的视图类型参数                                          | string  | 必须     | -                                                                                                |
| subViewFilter         | 右侧请求数据接口中 filter 参数，左侧树数据的字段参数，如 filter:['org_id','in',当前选中节点的 id]        | string  | 可选     | id                                                                                               |
| subViewFilterValue    | 右侧请求数据接口中 filter 参数，左侧树选中节点参数值的字段，如 filter:['id','in',当前选中节点的指定字段] | string  | 可选     | id                                                                                               |
| hasIcon               | 是否显示树节点右侧的图标按钮（添加、编辑、删除）                                                         | boolean | 可选     | false                                                                                            |
| showSubView           | 未选中树节点时，默认显示右侧视图                                                                         | boolean | 可选     | false                                                                                            |
| subViewFromDefault    | 将树的数据回填到详情页面的表单中，单个字段                                                               | string  | 可选     | -                                                                                                |
| subViewFormInitValues | 将树的数据回填到详情页面的表单中，多个字段                                                               | array   | 可选     | -                                                                                                |
| isUpdateTree          | 更新树数据，默认不更新树数据，标准页面中，删除表格数据时更新重新获取树数据                               | boolean | 可选     | -                                                                                                |
| isDrag                | 树宽度是否可进行拖拽调整                                                                                 | boolean | 可选     | -                                                                                                |

## subViewFormInitValues 配置参数

| 属性名       | 说明                                                                        | 类型    | 是否可选 | 默认值 |
| ------------ | --------------------------------------------------------------------------- | ------- | -------- | ------ |
| treeName     | 树对应字段                                                                  | string  | 必填     | -      |
| subformName  | 表单的回填字段                                                              | string  | 必填     | -      |
| label        | look，select 类型生效，下拉值的显示名                                       | string  | 可选     | -      |
| labelInValue | look，select 类型生效，下拉值的显示名跟值一致，label 配置之后优先使用 label | boolean | 可选     | -      |
